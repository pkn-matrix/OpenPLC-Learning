# Lesson 01 - Start/Stop Motor using Ladder Logic

## Objective

Design a simple motor start/stop circuit using Ladder Logic.

## Components

- Start Push Button (NO)
- Stop Push Button (NC)
- Motor Coil
- Seal-in Contact

## Logic

- Press Start → Motor turns ON
- Release Start → Motor remains ON
- Press Stop → Motor turns OFF

## Concepts Learned

- Normally Open Contact
- Normally Closed Contact
- Coil
- Seal-in (Latching) Logic
